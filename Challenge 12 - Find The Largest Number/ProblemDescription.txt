Programming Challenge: Find The Largest Number

Copied from Five programming problems every Software Engineer should be able to solve in less than 1 hour (problem 4): https://blog.svpino.com/2015/05/07/five-programming-problems-every-software-engineer-should-be-able-to-solve-in-less-than-1-hour

Write a function that given a list of non negative integers, arranges them such that they form the largest possible number.

For example, given [50, 2, 1, 9], the largest formed number is 95021.
